# cld0714 작업 기록

## 2026-07-14 CastleChat 버그 수정 (claude)

### 1. 텍스트 외 메시지는 공지 등록 차단
- 우클릭 메뉴에서 `messageType === 'TEXT'`이고 첨부가 없는 메시지에만 "공지" 버튼을 노출.
- 수정 파일 : `ChatBox.jsx` (컨텍스트 메뉴 렌더 조건 1곳).

### 2. room feed(공지/권한변경/초대/left/kick/ban)가 재입장 후 일반 메시지처럼 보이던 문제
- 구조 : feed는 DB에 `TEXT + "[ROOM_FEED]" prefix`로 저장된다(0714 codex 작업). 라이브 WS 이벤트는 FE가 SYSTEM 메시지로 합성해서 정상 표시되지만, 재입장 시 HTTP로 불러온 이력은 prefix가 그대로 노출되고 내 메시지 말풍선으로 렌더링됐다.
- BE 조회 SQL(`loadMessagesInRoom`)에는 이미 prefix→SYSTEM 변환 CASE가 있으나(오늘 18:46 수정), 화면에는 여전히 raw로 떴다 → **domain-service가 XML 수정 이후 재기동되지 않았을 가능성이 높다** (chatKafka.md J항목과 동일 패턴. MyBatis XML은 기동 시 1회만 파싱).
- 조치(FE, 서버 빌드 상태와 무관하게 동작하도록 단일 관문 정규화) :
  - `chatApi.js` `loadMessagesInRoomApi`에서 `[ROOM_FEED]` prefix 메시지를 `messageType: 'SYSTEM'` + prefix 제거로 정규화. 공지/권한변경/초대/left/kick/ban 전부 같은 prefix라 한 번에 해결.
  - `ChatBox.jsx`의 이전 메시지 페이징(중복 axios 호출)을 `loadMessagesInRoomApi`로 통일 → 정규화 관문 1곳.
  - `ChatList.jsx` 라이브 프리뷰에서도 prefix 제거.
- **재기동 필요** : domain-service (XML CASE 반영).

### 3. 친구 초대 패널이 채팅방 메뉴에 가려지던 문제
- 원인 : 도킹 모드 채팅방 메뉴 z-index 100, 초대 패널 40 + 위치가 메뉴와 겹침.
- 수정 : 도킹 모드에서 초대 패널을 메뉴 오른쪽 옆(`left: calc(min(380px, 76%) + 10px)`), z-index 110으로. `ChatBox.css`.

### 4. AI 추천 목록이 가려지던 문제
- 원인 : 추천 목록이 composer 영역 안에서 레이아웃을 밀며 늘어나 고정 높이에 잘림.
- 수정 : `.aiRecommendList`를 `.aiRecommendBox` 기준 `position: absolute; bottom: calc(100% + 6px)` 팝업으로 변경 — 입력창 위로 떠서 3개 전부 보임. `ChatBox.css`.

### 5. 채팅방 목록 unread가 읽은 방에서 다시 몇 개씩 뜨던 문제
- 원인 : 읽음 위치(LRM)는 Redis 선반영 + dirty flush로 DB에 늦게 반영되는데, 목록 조회 SQL(`getMyAllChatRooms`)은 DB의 `last_read_message_id`만 사용 → flush 전에 목록을 다시 열면 과거 LRM 기준 unread가 나옴.
- 수정(BE, domain-service) :
  - `ChatRoomListDTO`에 `myLastReadMessageId` 추가, 목록 SQL에서 함께 반환.
  - `RoomQueryService.getMyAllChatRooms`에서 방별로 Redis LRM을 조회해 DB LRM보다 크면 `countUnreadMessages`(신규, feed 제외)로 unread 재계산. unread 0인 방은 스킵.
- FE 보조 : feed로 인한 `CHAT_ROOM_UPDATED`는 unread 증가에서 제외(`ChatList.jsx`).
- **재기동 필요** : domain-service (+ common-contract 변경이므로 clean 없이 재빌드·재기동이면 충분).

### 수정 파일 요약
- FE : `chatApi.js`, `ChatBox.jsx`, `ChatBox.css`, `ChatList.jsx`
- BE : `RoomQueryService.java`, `RoomMapper.java`, `RoomMapperXml.xml`(domain-service), `ChatRoomListDTO.java`(common-contract)
